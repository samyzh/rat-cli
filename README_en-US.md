# rat-cli

The rat cli is a project scaffold/template manager based on github/gitlab

English | [简体中文](./README.md)
## Install

- use npm

```bash
npm i -g @samyzh/rat-cli
```

## init project

```bash
- rat init tpl/blk 
- rat tpl xxx
- rat blk xxx
- rat config set/get/del <k> [<v>]
- rat tpl/blk help
```
